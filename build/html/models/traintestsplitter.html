Train Test Splitter¶

class msc.traintestsplitter.TrainTestSplitter

This class implements several functions to stratify a dataset, either by creating splits from one datafile are by combining and reshuffling the existing train and test file

reshuffle(train_file_path: str, test_file_path: str, delimiter: str = ',', quotechar: str = '"', text_col_name: str = 'text', label_col_name: str = 'label', train_portion: float = 0.7) → Tuple[pandas.core.frame.DataFrame, pandas.core.frame.DataFrame]

This method can be used to ‘reshuffle’ existing train and test files when they already exist but a different distribution is required. This can be useful when the distribution of labels in one of the two files is significantly different from that of the other file.

Parameters

  • train_file_path – string specifying the the name of the file where the training set is located

  • test_file_path – string specifying the name of the file where the test set is located

  • delimiter – string specifying the delimiter used in the csv file, this is used by the csv reader

  • quotechar – string indicating the character used for quoting

  • text_col_name – string specifying the name of the header of the text column, defaults to ‘text’

  • label_col_name – string specifying the name of the header of the label column, defaults to ‘label’

  • train_portion – integer indicating the percentage of the total dataset that is used for training the test set size is set to (1-train_portoin), default is 0.7.

Returns

returns a tuple of two dataframes wher the first is the train dataframe and the second is the test dataframe

static stratify_file(file_name: str, delimiter: str = ',', quotechar: str = '"', text_col_name: str = 'text', label_col_name: str = 'label', train_portion: float = 0.7) → Tuple[pandas.core.frame.DataFrame, pandas.core.frame.DataFrame]

Method that can be used to read in an existing file and split it into train and test files with a stratification strategy to keep the distributions of labels between the two files as similar as possible.

Parameters

  • file_name – string specifying the path and name where the datafile is located

  • delimiter – string specifying the delimiter used in the csv file, this is used by the csv reader

  • quotechar – string indicating the character used for quoting

  • text_col_name – string specifying the name of the header of the text column, defaults to ‘text’

  • label_col_name – string specifying the name of the header of the label column, defaults to ‘label’

  • train_portion – integer indicating the percentage of the total dataset that is used for training the test set size is set to (1-train_portoin), default is 0.7.

Returns

returns a tuple of two dataframes wher the first is the train dataframe and the second is the test dataframe
