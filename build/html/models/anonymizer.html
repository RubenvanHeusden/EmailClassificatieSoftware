Anonymizer¶

class msc.anonymizer.Anonymizer(replacement_string: str = '')

This class implements a automatic anonymization algorithm for Dutch text using the spaCy Named Entity Recognition algorithm

nlp_model

the subset of spaCy models for the language passed in as parameters, the Dutch version is used in this class

convenience method to anonymize all the text entries in a file.

Parameters

  • file_name – string specifying the name of the csv file which contains the data to be anonymized.

  • delimiter – the delimiter used for the reading of the csv file, default is ‘,’

  • quotechar – the quotation character used by the csv reader, default is ‘”’

  • text_col_name – string signifying the name of the column in the csv file containing the text that is te be anonymized.
