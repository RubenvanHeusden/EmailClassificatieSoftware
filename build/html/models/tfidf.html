classify_from_file(classification_data_path, text_col_name: str = 'text', delimiter=',', quotechar='"') → List[Any]¶

classifies examples from a csv file with the trained classifier, where the column with the text to be classified is selected with the text_col_name parameter

Parameters

  • classification_data_path – string specifying the file that contains the texts to be classified by the classifier

  • text_col_name – string specifying the name of the column containing the mails in the csv file

  • delimiter – string specifying the delimiter of the csv file, defaults to “,”

  • quotechar – specifying the character as quotechar in the csv reader. defaults to “

Returns

A list of predictions with the same length as the number of data points in the file, where each entry in the list is a prediction of the model

score(test_data_path, text_col_name, label_col_name, delimiter, verbose, class_averaging)¶
Parameters

  • test_data_path – string specifying the file that contains the data points that have to be scored

  • text_col_name – string specifying the name of the column containing the mails in the csv file

  • label_col_name – string specifying the name of the column containing the labels of the mails in the csv file

  • delimiter – string specifying the delimiter of the csv file, defaults to “,”

  • quotechar – specifying the character as quotechar in the csv reader. defaults to “

  • verbose – integer specifying the verbosity of the returned score, below are the possible options: 1: return only accuracy 2: return overall accuracy, precision, recall and f1 3: return detailed sklearn classification report Because the dataset is multiclass and there is possible class imbalance, weighted versions of all classification scores are used

  • class_averaging – specifies the kind of averaging to use for the scoring in a multiclass classification setting. see Sklearn documentation for more details on the options.

Returns

scores indicating the performance of the model, depends on level of verbosity set.
